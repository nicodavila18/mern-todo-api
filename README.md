# Gestor de Tareas Full-Stack con Node.js y MongoDB

## Motivación del Proyecto
Este proyecto se está desarrollando para dominar la creación de una API RESTful y el ciclo CRUD (Create, Read, Update, Delete) de principio a fin, como parte de mi preparación profesional.

## Stack Tecnológico Inicial
- Backend: Node.js (Express)
- Base de Datos: MongoDB (Mongoose)
- Frontend: JavaScript (Vanilla JS), HTML, CSS (Tailwind CSS)
- Control de Versiones: Git
